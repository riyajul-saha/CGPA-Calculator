# CGPA Calculator

## Description
The CGPA Calculator is a tool designed to calculate the Cumulative Grade Point Average (CGPA) based on various grading systems. It provides students with the ability to monitor and manage their academic performance effectively.

## Features
- Easy-to-use interface
- Supports multiple grading systems
- Detailed reporting and analysis
- Customizable grading scales

## Installation Instructions
1. Clone the repository:
   ```sh
   git clone https://github.com/riyajul-saha/CGPA-Calculator.git
   ```
2. Navigate to the directory:
   ```sh
   cd CGPA-Calculator
   ```
3. Install the required dependencies[Before run please confirm that Python already installed in your device]:
   ```sh
   pip install -r requirements.txt
   ```
